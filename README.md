# Sanos y Salvos -  MS Publicaciones

Este microservicio fue generado utilizando un Arquetipo Maven para garantizar su estructura estándar.

## Requisitos previos
- Java 17
- Maven

## Instalación y Ejecución
1. Abre una terminal en la raíz de este proyecto.
2. Compila y descarga dependencias: `./mvnw clean install` (o `mvn clean install` si tienes Maven global).
3. Levanta el servicio: `./mvnw spring-boot:run`
